W31
Baseline model — simple ML, no deep learning
▶
Deliverables
  • Encode protein sequences (e.g. amino acid composition features)
  • Train logistic regression + random forest baselines
  • Record metrics: accuracy, F1 per class, confusion matrix
  • This is your benchmark — deep model must beat this to mean anything
W32
Implement paper architecture in PyTorch
▶
Deliverables
  • Implement the model architecture described in the paper
  • Sequence encoding → embedding → model layers → output
  • Training loop with proper logging (loss, accuracy per epoch)
  • Don't try to improve yet — reproduce first
W33
Train + evaluate + debug
▶
Deliverables
  • Train model — expect it to fail first time. That's normal.
  • Debug: is loss decreasing? Is model overfitting? Is learning rate right?
  • Aim to match paper's reported accuracy within ~5% margin
  • Log all experiments — what you tried, what changed
W34
Analyze failures + write findings
▶
Deliverables
  • Which classes does the model confuse most? Why?
  • Where does your implementation differ from the paper?
  • Write 300-word analysis in notebook — what failed and the hypothesis for why
  • This analysis = the most valuable part of the project
W35
Polish repo — README, requirements, usage
▶
Deliverables
  • README.md: problem statement, approach, results, how to run
  • requirements.txt with pinned versions
  • Clear notebook structure — someone unfamiliar should be able to follow it
  • All raw data excluded from git, instructions for downloading included
